# 🤖 Dialogflow AI Chatbot

An intelligent conversational chatbot built using **Google Dialogflow** that understands natural language queries and provides automated responses. This project demonstrates the implementation of Conversational AI for customer support, financial guidance, FAQs, and other interactive applications.

---

## 📌 Project Overview

The Dialogflow AI Chatbot leverages Google's Natural Language Understanding (NLU) capabilities to recognize user intents, extract entities, and generate meaningful responses in real time. Dialogflow enables developers to create scalable conversational experiences for websites, mobile applications, and messaging platforms.

## 🚀 Getting Started

### Prerequisites

* Google Account
* Dialogflow ES/CX Access
* Google Cloud Project
* Internet Connection

### Setup Steps

1. Sign in to Dialogflow Console.
2. Create a new Agent.
3. Configure Default Language and Time Zone.
4. Create Intents and Training Phrases.
5. Define Entities if required.
6. Enable Fulfillment/Webhooks.
7. Test the chatbot using the Dialogflow simulator.
8. Deploy to your preferred platform.

# 💰 Financial Advisor Chatbot using Dialogflow & Kommunicate

## 📌 Project Overview

The **Financial Advisor Chatbot** is an AI-powered virtual assistant built using **Google Dialogflow** and integrated with **Kommunicate**. The chatbot helps users make informed financial decisions by providing guidance on budgeting, loans, insurance, investments, tax savings, and key financial ratios.

The bot uses intent-based conversation flows to understand user queries and deliver relevant financial advice in real time.

---

## 🚀 Features

* 📊 Budget Planning Advice
* 💳 Credit Score Guidance
* 🏦 Loan Recommendation Assistance
* 🛡️ Insurance Advisory
* 📈 ROI (Return on Investment) Calculator
* 📉 NPV (Net Present Value) Calculator
* 📈 IRR (Internal Rate of Return) Calculator
* ⚖️ Debt-to-Equity Ratio Analysis
* 📊 Current Ratio Analysis
* 💰 Break-Even Point Calculation
* 🧾 Tax Saving Suggestions
* 🤖 Fallback Handling for Unknown Queries
* 👋 Welcome and Greeting Intents

---

## 🛠️ Technologies Used

* **Google Dialogflow ES**
* **Kommunicate**
* **Natural Language Processing (NLP)**
* **Intent-Based Conversation Design**
* **Webhook Integration (Optional)**

1. User enters a financial query.
2. Dialogflow identifies the intent.
3. Appropriate financial advice or calculation is triggered.
4. Response is displayed through Kommunicate.
5. Unknown queries are handled by the Fallback Intent.
